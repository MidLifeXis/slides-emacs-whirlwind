* What is Emacs?

  - An editor
  - A programming environment
  - A LISP interpreter wrapped in a REPL
  - An operating system
  - A customizable development environment
  - Cross platform
  - Everything, including the kitchen sink
  - To some, a religion

* What can Emacs do?

  - Communication (IRC, email, web, twitter)
  - web / gopher / ftp
  - Data manipulation
  - Syntax Highlighting
  - Compilation
  - Integrate with other systems
